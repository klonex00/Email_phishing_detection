# Model Files

The large model files are not included in this repository due to size constraints.

## Required Models

### Classical Models
- `classical/fuse_lr.joblib`
- `classical/url_gbm.joblib`

### Text Model
- `text/text_model_final/` (entire directory)

## Setup Instructions

If you cloned this repository, you'll need to obtain the model files separately:

1. Contact the repository owner for access to the model files
2. Place them in the appropriate directories as shown above
3. Ensure all files are in place before running the application

## Alternative: Train Your Own Models

You can also train your own models using the training scripts (if provided) or your own dataset.
